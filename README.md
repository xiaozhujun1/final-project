# TodoList 待办清单小程序

一个简洁实用的微信小程序待办清单应用，帮助你高效管理日常任务。

## 功能特性

- **任务管理**：创建、编辑、删除待办事项
- **分类管理**：支持工作、生活、学习、其他四种分类
- **优先级设置**：高、中、低三种优先级
- **截止日期**：设置任务截止日期，过期任务自动提醒
- **任务备注**：添加详细备注信息
- **任务搜索**：按标题或备注搜索任务
- **筛选排序**：支持按状态筛选和多种方式排序
- **数据统计**：查看任务完成情况和分类统计
- **数据备份**：支持数据导出和导入，防止数据丢失

## 项目结构

```
final-project/
├── pages/
│   ├── index/          # 首页（任务列表）
```

## 技术栈

- 微信小程序原生开发（MINA框架）
- 本地存储（wx.setStorageSync）

## 开发环境

- 微信开发者工具
- Node.js（可选）

## 使用说明

1. 打开微信开发者工具
2. 导入项目文件夹
3. 点击"编译"按钮运行

## 页面说明

### 首页（index）
- 展示所有待办任务列表
- 支持按全部/待完成/已完成筛选
- 支持按创建时间/优先级/截止日期排序
- 搜索任务功能
- 点击任务标记完成/未完成
- 长按任务弹出操作菜单

### 添加任务（add）
- 输入任务标题
- 选择任务分类
- 设置优先级
- 设置截止日期（可选）
- 添加备注（可选）

### 编辑任务（edit）
- 修改任务信息
- 更新分类、优先级、截止日期和备注

### 数据统计（stats）
- 总体任务完成情况
- 各分类完成率统计
- 优先级分布统计
- 数据备份入口

### 数据备份（backup）
- 导出任务数据（JSON格式）
- 导入任务数据
- 清空所有数据

## 数据存储

任务数据使用微信小程序本地存储 API `wx.setStorageSync` 存储在 `todos` 键下，格式如下：

```json
{
  "title": "任务标题",
  "category": "工作",
  "priority": "高",
  "completed": false,
  "createTime": 1625097600000,
  "hasDeadline": true,
  "deadline": "2026-06-30",
  "remark": "备注信息"
}
```
